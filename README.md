# Student CRUD App

A clean and responsive React application to create, read, update, and delete student records.

## Features
- Add new student records
- View all students in a structured table
- Edit existing student details
- Delete student records
- Search and filter students by key fields
- Dashboard-style stats (for example: total students, average GPA)
- Responsive design for desktop and mobile

## Tech Stack
- React
- Vite
- JavaScript (ES6+)
- CSS3

## Installation
```bash
git clone <your-repo-link>
cd student-crud
npm install
npm run dev
```

Open the local URL shown in terminal (usually `http://localhost:5173`).

## Usage
1. Fill out the form and add a student record.
2. Use **Edit** to update any student details.
3. Use **Delete** to remove a record.
4. Use search/filter to quickly find matching students.

## Build for Production
```bash
npm run build
```

The optimized production files will be generated in the `dist` folder.
